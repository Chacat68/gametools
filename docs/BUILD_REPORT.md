## 功能模块

### 1. 越南文检测
- 检测表格文件中的越南文内容
- 支持Excel和CSV文件格式
- 批量扫描和精确定位功能

### 2. JSON格式检测工具
- 检测JSON文件中text字段的格式一致性
- 自定义检测字段名
- 生成详细的检测报告

### 3. Excel数据处理工具
- 根据指定列对Excel数据进行分组和处理
- 智能分组Excel数据，支持多文件输出
- 灵活配置分组列和工作表前缀

### 4. 翻译提取工具
- 批量提取Excel文件中的文本内容
- 智能文本识别和分类
- 支持多种文本类型过滤
