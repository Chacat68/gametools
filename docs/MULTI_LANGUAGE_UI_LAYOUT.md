## 🎯 操作流程图

```
开始
  │
  ├─> 1. 配置语言路径
  │     ├─ 中文版: 选择文件/目录
  │     ├─ 越南文版: 选择文件/目录
  │     ├─ 英文版: 选择文件/目录
  │     ├─ 韩文版: 选择文件/目录
  │     └─ 日文版: 选择文件/目录
  │
  ├─> 2. 设置输出目录
  │     └─ 选择保存提取结果的位置
  │
  ├─> 3. 配置选项
  │     ├─ 递归扫描子目录
  │     └─ 选择文本类型
  │
  ├─> 4. 预览文件（可选）
  │     └─ 查看将要处理的文件列表
  │
  ├─> 5. 开始提取
  │     ├─ 处理中文版
  │     ├─ 处理越南文版
  │     ├─ 处理英文版
  │     └─ ...
  │
  └─> 6. 查看结果
        ├─ 分语言统计
        ├─ 总体统计
        └─ 打开输出目录
```

## 🔧 按钮功能说明

### 浏览按钮
- **功能**: 选择文件或目录
- **交互**: 弹出选择对话框
  - 是(Y): 选择单个Excel文件
  - 否(N): 选择目录

### ✕ 按钮
- **功能**: 清除已配置的路径
- **快速**: 一键清空对应语言的配置

### 开始提取
- **功能**: 执行多语言文本提取
- **验证**: 自动检查路径有效性
- **处理**: 后台线程，不阻塞界面

### 清空结果
- **功能**: 清空结果显示区域
- **用途**: 开始新的提取任务前清理界面

### 预览文件
- **功能**: 查看将要处理的文件列表
- **显示**: 分语言展示文件统计

## 💾 使用技巧

### 技巧1: 快速配置
1. 先配置第一种语言
2. 系统自动设置输出目录
3. 继续配置其他语言

### 技巧2: 混合使用
- 主要语言选择目录（批量处理）
- 更新语言选择文件（单个处理）

### 技巧3: 输出管理
- 使用统一的输出目录
- 每种语言自动创建子文件夹
- 便于版本对比和管理

### 技巧4: 进度监控
- 实时查看日志输出
- 彩色图标区分状态
- 进度条显示整体进度

## ⚡ 快捷操作

### 键盘快捷键
- `Tab`: 在输入框间切换
- `Enter`: 确认对话框
- `Esc`: 取消对话框

### 鼠标操作
- 单击"浏览": 选择路径
- 单击"✕": 清除路径
- 双击结果区: 全选文本
