## 🎉 重大更新

### 翻译提取工具 v2.0 - 全新输出格式

本次更新对翻译提取工具进行了重大重构，优化了输出格式和用户体验。

## ✨ 新增功能

### 1. 统一汇总表格输出
- **新格式**: 生成单一 `翻译提取汇总.xlsx` 文件
- **表格结构**: 
  - `name` - 文件名（不含扩展名）
  - `num` - Excel物理位置（如F7）
  - `cn` - 中文文本
  - `vn` - 越南文文本  
  - `en` - 英文文本
  - `th` - 泰文文本（新增）

### 2. 语言配置调整
- ✅ **中文版**: 作为基准语言（必选）
- ✅ **越南文版**: 默认语言
- ✅ **英文版**: 额外支持
- ✅ **泰文版**: 新增支持
- ❌ **删除**: 韩文版、日文版

### 3. 智能匹配机制
- 以中文版为基准，自动匹配其他语言对应位置的文本
- 精确定位：记录Excel物理位置（如F3），便于查找原文
- 跨语言文本对齐：相同位置的文本自动关联

### 4. 美化表格格式
- 标题行：蓝色背景（#4472C4）+ 白色粗体字
- 自动列宽调整：
  - name列：25字符
  - num列：10字符
  - 语言列：40字符
- 专业的Excel样式，便于阅读和编辑

1. 下载 `gametools_v1.21.0.exe`

### 方式2: 便携版
1. 下载 `gametools_v1.21.0_便携版.zip`


## 🚀 使用方法

### 翻译提取工具使用步骤

1. **启动程序**
   - 双击 `gametools_v1.21.0.exe`

2. **切换到翻译提取页签**

3. **配置语言路径**
   - **中文版**: 必选，作为基准
   - 越南文版: 可选
   - 英文版: 可选
   - 泰文版: 可选
   - 每种语言可选择单个Excel文件或包含多个文件的目录

4. **设置输出目录**（可选）
   - 如未设置，默认使用中文版的目录

5. **点击"开始提取"**

6. **查看结果**
   - 提取完成后，在输出目录查看 `翻译提取汇总.xlsx`

## 💡 使用技巧

### 文件组织建议

```
translations/
├── cn/          # 中文版目录
│   ├── config_table.xlsx
│   └── skill_table.xlsx
├── vn/          # 越南文版目录
│   ├── config_table.xlsx
│   └── skill_table.xlsx
├── en/          # 英文版目录
│   └── config_table.xlsx
└── th/          # 泰文版目录
    └── config_table.xlsx
```

### 注意事项

1. **中文版必须配置** - 作为基准语言
2. **文件名应保持一致** - 各语言版本使用相同的文件名（不含语言后缀）
3. **工作表结构应一致** - 保持相同的行列结构
4. **支持文件格式** - .xlsx（推荐）、.xls
